# Markdown ile Teknik Dökümantasyon

Markdown; başlık, liste, tablo, kod blokları gibi öğeleri kolayca yazmanıza olanak tanır.

## Örnek Tablo
| Adım | Açıklama |
|------|----------|
| 1    | Depoyu klonlayın |
| 2    | Dosyaları düzenleyin |
| 3    | GitHub'a gönderin |

## Kod Bloğu
```bash
git add .
git commit -m "Yeni içerik eklendi"
git push
```
